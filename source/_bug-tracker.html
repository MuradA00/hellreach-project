Bug tracker

Bug reporting nodes

When reporting a bug, please make sure there are no duplicates and please include methods to reproduce the issue you have encountered. Please include your in-game character name and Discord so we can communicate with you if needed and for the appropriate reward for reporting an existing issue.

Status Definitions
Fixed

The issue has been fixed and/or implemented and should be pushed live shortly if it hasn't been already.

Confirmed

We’ve reviewed the bug report and found the source of the error. Fix imminent.

Pending

We have not yet reviewed the report.

Testing fix

We have not yet reviewed the report.

Invalid/duplicate

Reports that we cannot reproduce or need more information. In this case just add a comment and we will review it again.
